This lab contains a set of experiments that compare a test sequence generation library against data found in related works.

The lab is an instance of the LabMate environment, a framework for producing self-contained sets of input data and experimental scripts. For more information on how to use the environment, please refer to the help page.

Structure of the lab

The lab consists of two parts:

  1. In the first part, test sequences are generated from a set of finite-state machines. Some of these FSMs are extracted from past literature, while others are original to this work. You can see the complete list of all the FSMs included in the lab, along with the source files and pictures for each.
  2. In the second part, test sequences are generated from a set of Linear Temporal Logic formulæ.
